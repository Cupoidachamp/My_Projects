# AWS SmartGear Retail Data Pipeline Project

This project demonstrates the implementation of a simple data engineering pipeline using AWS services.

## Services Used
- Amazon S3
- AWS Glue
- Amazon Athena
- Amazon Redshift
- IAM

## Project Overview
The objective of this project was to build a basic data pipeline for processing retail sales data.

The pipeline performs the following steps:

1. Raw data is stored in Amazon S3.
2. Data is processed using AWS Glue.
3. Data is queried using Amazon Athena.
4. Aggregated data is stored and analyzed using Amazon Redshift.
5. A Glue job automates the pipeline by copying files from the raw layer to the landing layer.
